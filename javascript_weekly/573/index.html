    要避免的 TypeScript 功能？ (opens new window) — 还记得十年前 Gary Bernardt 的 Wat (opens new window) 视频吗？他创建了 Execute Program (opens new window) 开发教育平台，现在反思 TypeScript 中需要认真思考的四件事：枚举、命名空间、装饰器和私有。这是 自以为是 的。

    使用 structuredClone() 深度复制对象 (opens new window) — 当谈到复制 JavaScript 对象时，Axel 博士会着眼于新的东西：structuredClone。大多数浏览器的尖端版本以及 Node.js 和 Deno 都支持它，是时候考虑一下了。

    David Khourshid 的 XState 状态机新课程 (opens new window) — 如果您构建大型 JavaScript 应用程序并希望避免遇到奇怪的边缘情况，您将需要学习状态机。花时间清晰而稳健地为应用程序逻辑建模。

    JavaScript 风格的 Monad 简介 (opens new window) — “我知道很多人会翻白眼”，Kyle 谈到另一个 monad 解释器时说，但是.. 如果你喜欢 Kyle 与 You Don't Know JS (opens new window) 的作品，或者你想要一些面向 JavaScript 的东西，这个是给你的。

    Remix vs Next.js (opens new window) — Remix (opens new window) 是 JavaScript 全栈框架的新星，因此会很自然地将它与 Next.js（甚至 Ruby on Rails）等其他东西进行比较。请记住，这种比较来自 Remix 团队本身，不过写得很好，当然，同时促进了 Remix 方法。

    Deno 1.18 发布:（其他）现代 JS/TS 运行时 (opens new window) — Deno 继续向前迈进。 1.18 有全面的 Web Cryptography API (opens new window)、将 V8 升级到 9.8 版、缩短启动时间等方面的巨大努力。
