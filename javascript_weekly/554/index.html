    JavaScript Weekly #554

    2022/2/1

    Visual Studio Code 2021 年 8 月更新 (opens new window) — 新功能包括自动语言检测、内置括号着色（通过 editor.bracketPairColorization.enabled 启用）、改进了终端字形渲染、JS/TS 的内联提示参数名称和类型，也改进了异步函数和 Node.js 内部的调试步骤。

    ES2022 特性：类静态初始化块 — 这个提议处于包含在 ES2022 的第 4 阶段，看起来也不觉得 JavaScript 的这种写法偏向于 Java。

    💡 像这样 ↑ ？ Hemanth HM 汇总了 TC39 第 85 次会议 (opens new window) 在未来（潜在）JavaScript 新特性的更新，包括 hack pipe (opens new window) 运算符和 Array.fromSync (opens new window)。

    使用 Sourcegraph 在开源宇宙中搜索 (opens new window) — Sourcegraph 可以在 GitHub 和 GitLab 的超过 100 万个开源存储库以及您自己的公共和私有（公共测试版）存储库上使用，无需安装。 Sourcegraph 让搜索、发现和阅览开源项目变得更加容易。

    Electron 14.0.0 发布 (opens new window) — 流行的跨平台桌面应用程序框架的最新版本已发布。预计稳定发布还有八周，因此 Electron 15 将在三周后发布。现在您可以获得 Chromium 93、Node.js 14.17 和 V8 9.3 支持、实验性 cookie 加密支持和一些删除（例如作为 远程模块 (opens new window)）。

    Neutralino 2.7：便携、轻量级的桌面应用程序框架 (opens new window) — 想想 Electron，它没有在每个应用程序中都包含 Chromium 发行版，而是使用本地平台的内置“webview”。也就是说它会更小、更快，虽然您得必须了解每个 webview 的兼容性。

    # 🛠  代码 & 工具

    TypeIt：一个多功能的假装打字效果实用程序 (opens new window) — 自称是 “地球上最通用的 JavaScript 打字机实用程序”。有易用的 API，但请注意，虽然它对个人/OSS 使用是免费的，但您需要支付（少量）商业许可证。

    Stitches 1.0：现代样式库 (opens new window) — 另一种 CSS-in-JS 解决方案，它提供了高性能、SSR、主题、关键路径 CSS、直观的 API 等等。

    使用开源 Unikernel 要比 Linux 运行得更快、更安全 (opens new window) — 无需操作和云锁定即可部署。在 Google 上以 2 倍的速度运行。在 AWS 上以 3 倍的速度运行并在 10 秒内完成部署。

    smartcrop.js：内容感知图像裁剪 (opens new window) — 给定一张图像，它可以对图像的 “最佳” 部分进行裁剪，且运行在浏览器和 Node.js 中。

    Roadroller：JavaScript 代码的重量级 “破碎机” (opens new window) — 该工具是为了生成演示风格的编码而不是构建应用程序，其目标是将 JavaScript 代码压缩/缩小到尽可能小的尺寸，GitHub 存储库 (opens new window)。

    Crank.js：使用 Functions、Promises 和 Generators 编写 JSX 驱动的组件 (opens new window) — 您可能喜欢直接在 JavaScript 中编写类似 HTML 的代码的常用 JSX 写法，但是所有组件也都只是 Functions 或 Generators。 查看一些示例代码 (opens new window)。

    在 SnykCon 上宣布 JavaScript 安全会议 | 免费注册 (opens new window)

    parse-domain：将主机名拆分为多个部分 (opens new window) — 给定主机名，提取子域、域和顶级域，包括 公共后缀列表 (opens new window) 中的 “有效” 顶级域。

    Mafs：用于数学可视化的 React 组件 (opens new window) — 一套较完善的的复杂数学可视化组件。它有一套不错的 “你需要知道的” 指南 (opens new window) 支持，其中包括一个可以让您计划自己的 [亚轨道太空飞行](suborbital spaceflight)的指南。

    bundle：一个快速检查 npm 包大小的检查器 (opens new window) — 输入包名称，然后点击 “run” 按钮，此工具将为您提供压缩、捆绑和展示 gzip 压缩后的包大小。
