    Ecma International Approves ECMAScript 2022, But What’s New? (opens new window) — Thanks Ecma – ECMAScript 2022 is now an approved standard. (opens new window) JavaScript’s resident doctor brings us up to speed with what this means in practice, as well as answering a few common questions. You’ll already be using some of these features – this is mostly about tying up the formalities. On to ES2023..

    Dr. Axel Rauschmayer

    ❓ If you're unfamiliar with ECMAScript as a term, it's the name of the generic, standardized language behind JavaScript implementations, as JavaScript is technically a trademark of the Oracle Corporation. Longer explanation. (opens new window)

    In Defense of Blocks to Create Localized Scope (opens new window) — Kyle Simpson (of YDKJS fame) pops up to give his thoughts on a recently popular article about using ‘block statements’ (opens new window) to group code together which attracted some criticism. In short, Kyle’s a big fan of the approach.

    Kyle Simpson

    New Course on Web Components (opens new window) — "Web Components are one of the most exciting new technologies on the web platform today." ~Dave Rupert. Learn to create your own custom web components using JavaScript and the Lit library in this course.

    Frontend Masters sponsor

    When You Should Prefer Map Over Object In JavaScript (opens new window) — Map objects are (nearly) universally available in JS implementations now and present some benefits to using regular Objects.

    Zhenghao He

    What the React Team is Working On Now (opens new window) — With the React 18 launch behind them, the React team has turned their attention to the future and this post gives some detailed hints of what’s going on. It’s not a road map, though, and they stress that some of what’s discussed many not even materialize as-is.

    The React.js Team

    • The beta of TypeScript 4.8 (opens new window) is here, complete with performance improvements and, as usual, a handful of intricately geeky type inference enhancements.

    • 💰 The company behind Deno (opens new window) has raised $21m in a series A funding round. (opens new window) The money will help them take the V8 isolates-based Deno Deploy (opens new window) service to the next level.

    • 🤭 Why choose one JavaScript CDN when you can make them all fight it out? (opens new window)

    • Vue 2.7 is in beta. (opens new window) "2.7," you say? Some Vue 3 features have been backported and it's aimed at folks who need to stay on Vue 2.x due to dependencies.

    • 🤖 GitHub Copilot, an 'AI pair programmer' of sorts, is now generally available (opens new window) after a long preview phase. It's pretty effective, if controversial. (opens new window) In related news, Amazon has unveiled its own attempt called Amazon CodeWhisperer (opens new window) which, too, supports JavaScript.
