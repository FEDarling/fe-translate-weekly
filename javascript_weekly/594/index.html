    AbortController is Your Friend (for Cancelling Async Tasks and More) (opens new window) — AbortController (opens new window) provides a way to abort web requests at any point without waiting for a response, but it’s possible to apply its handy mechanism to other use cases, as shown here.

    Sam Thorogood

    Microvium: A Tiny JavaScript Engine for Microcontrollers (opens new window) — How tiny? Try 8.5 KB and needing just 34 bytes of RAM at idle. Despite neat projects like Espruino (opens new window), JS has never been a natural fit for constrained environments but Microvium, Elk (opens new window) and low.js (opens new window) are working hard at the problem. Here’s how Microvium works (opens new window) and the obligatory GitHub repo. (opens new window)

    Michael Hunter

    (opens new window)

    Couchbase Capella DBaaS: Store in JSON, Access with SQL (opens new window) — Build faster with in-memory performance, automatic replication and scaling. Try it now for free and be live in under three minutes

    Couchbase sponsor

    Vitest: A Vite-Powered 'Blazing Fast' Unit Testing Framework (opens new window) — Brings a lot of cool stuff into one place and really is quick – if you want near instant feedback when you change your code, check it out. You get Vite’s transformers, resolvers, and plugins, snapshot testing, native code coverage, component testing, an instant watch mode, and more.

    Vitest

    • ⭐️ Last week, Vue.js's creator gave ▶️ a 'State of the Union'-style address (opens new window) on the project (plus Vite) for Vue.js Amsterdam.

    • 👾 OneJS (opens new window) is an intriguing JavaScript-based scripting engine for building UIs in the Unity game engine. There's some example code (opens new window) of using it to create a UI on top of a game.

    • Someone scraped 1 million top domains to analyze what JS libraries they're relying upon. There's a lot of data (opens new window) or just enjoy this summary graphic (opens new window) showing that jQuery leads the way! Ken Wheeler's Slick (opens new window) also does very well. WordPress seems to have a big influence here.

    • 📆 Sails (opens new window) is a MVC-oriented webapp framework for Node and an online Sails event, Sailsconf (opens new window), is taking place on YouTube (opens new window) on June 22-24 (next Weds-Fri).

    • The end-of-life date for Node.js 16 (opens new window) has been brought forward to September 2023 to coincide with the EOL of OpenSSL 1.1.1.

    # 🛠 Code & Tools

    (opens new window)

    Component Encyclopedia: A Catalog of 5000+ UI Components (opens new window) — Storybook (opens new window), a UI component devtool, has unveiled a visual catalog (opens new window) of 5,000+ components (mostly React, but some Vue and Web Components) – you don’t need to be a Storybook user as links to repos and demos are provided.

    Dominic Nguyen (Storybook)

    ow 1.0: Function Argument Validation for Humans (opens new window) — A fluent API to define constraints for function arguments (e.g. ow(input, ow.string.minLength(5))) and get nice error messages on failure. Now a pure ESM package. GitHub repo. (opens new window)

    Sindre Sorhus

    zx v7.0: Better Shell Scripting with JavaScript (opens new window) — If writing JavaScript sounds better than wrestling with bash, Perl or Python, zx is for you. v7.0 is rewritten in TypeScript, has an echo (opens new window) function for printing the output of other scripts, within (opens new window) for creating new async contexts, and a new interactive REPL mode (behind -i).

    Google

    Fx 24.0: A Command-Line JSON Processing Tool (opens new window) — If you’ve got some JSON you want to slice and dice, Fx is for you. It was recently rewritten in Go from its original JavaScript but you can still write reducers in JavaScript (or Ruby, or Python) with it. A handy tool.

    Anton Medvedev

    Studio 3T Manages Your Data, While You Wrestle Your App into Shape (opens new window) — Studio 3T's full suite of MongoDB tools lets you query quicker, develop with data even faster and turn queries into code automatically. Try it free for 30 days - no credit card needed.

    Studio 3T sponsor

    Moon: A New Build System for the JavaScript Ecosystem (opens new window) — Built in Rust for performance, Moon appears to be focused on features for larger projects that may have lots of dependencies, developers, and processes to balance. It’ll be interesting to see this one develop further.

    Miles Johnson

    public-ip 6.0: Quickly Get Your Public IP Address (opens new window) — Works in both Node and the browser and uses different approaches on each. v6 lets you choose between IPv6 and IPv4.

    Sindre Sorhus
