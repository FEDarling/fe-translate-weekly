    Roll Your Own JavaScript Runtime (opens new window) — Bun, Rhino, Deno, Node, and now Blueboat.. is creating your own JavaScript runtime environment the new “hello world”? 😆 Not quite, but this is a fun walk through the bare basics of the process, and you’ll have a Rust, Tokio and V8 powered system of your own by the end of it.

    Bun: A Complete Overhaul of the JavaScript Ecosystem? (opens new window) — Continuing on the JavaScript runtime theme, Forrest takes a thoughtful look at Bun, its promises, and potential. While it’s “still not quite ready for the workplace,” he concludes there are clearly good times ahead.

    Forrest Allison
