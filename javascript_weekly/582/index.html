    Parcel v2.4.0 Released: The Zero Config Build Tool (opens new window) — Parcel’s new CSS parser and compiler (opens new window) – Parcel CSS – now makes it into the main Parcel build tool as the default CSS transformer and minifier. There is more, of course: CSS Color Level 4 support, cascade layers, improved JS dead code elimination, and support for class static initialization blocks.

    Devon Govett and Team

    RFC: Intent to Ship React 18 (opens new window) — This RFC covers the new features and changes coming to React 18. While React 18 is very close, the team would like to get ‘one final round of broad public feedback’ (which you can submit here (opens new window)) before shipping.

    React Community

    Testing Your Web Apps with Cypress (opens new window) — Join Steve Kinney on this detailed course covering Cypress: A fast, easy way to write end-to-end tests for your client-side apps. Learn about the selector engine, test runners, assertion library, and more.

    Frontend Masters sponsor

    The Tao of Node: On Design, Architecture and Best Practices (opens new window) — A developer has boiled down all of the hard-earned best practices and gut feelings he’s picked up for developing high-quality Node apps over time.

    Alex Kondov

    ✉️  This Week's Issue of React Status: It's Where All the React Links Go (opens new window) — If you work with React at all, be sure to check out our sister weekly: React Status. This week saw a lot of solid releases including that of Turnstone (opens new window), a new React search box component, and Liqvid 2.1 (opens new window), a system for creating interactive videos with React.

    Cooperpress
