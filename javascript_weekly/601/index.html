    JavaScript Weekly #601

    2022/8/18

    (opens new window)

    JS1024 2022 Competition Winners (opens new window) — JS1024 is a ‘code golfing’ contest where you get 15 days to create a JavaScript or GLSL program within 1024 bytes. This results in lots of creative ideas, complete with fantastic (non-minified) source to check out. The winning JS entry is this quirky typing game (opens new window) and the winning shader-based entry is a 3D tunnel experience. (opens new window) There’s much to learn here as these folks have used every trick in the book, including compressing 1900+ English words in a heavily commented entry that uses (opens new window) Brotli and a specially crafted WOFF2 font file(!)

    JS1024

    Docusaurus 2.0 Released (opens new window) — Docusaurus powered sites are everywhere in the JS space (e.g. the Babel docs (opens new window)) and after four years of effort, version 2.0 (opens new window) of the popular React-powered documentation-focused static site generator is here. Rebuilt from scratch, it adds MDX (opens new window) support, a new plugin system, theming, dark mode, SPA navigation, offline support, and more.

    Meta (Facebook)

    (opens new window)

    Announcing EdgeDB 2.0 (opens new window) — Try the database that makes ORMs obsolete. EdgeDB’s TypeScript client has the simplicity of an ORM but the power of a full-fledged query language. And with the release of EdgeDB 2.0 it’s better than ever—read the announcement for details.

    EdgeDB sponsor

    Electron 20: The Cross Platform Desktop App Framework (opens new window) — 20 sounds like a big, round release number but it’s mostly a typical release. Chromium is bumped up to v104, V8 to 10.4, and Node to 16.15.0. Windows gains an ‘immersive dark mode’ and Electron windows can now ‘float’ over full-screen apps panel-style (opens new window).

    Charles Kerr
