    # 2021 年最棒的 JavaScript Weekly

    这是今年的最后一篇（我们在 2022 年 1 月 7 日回来）所以我们回顾了 2021 年的一些最受欢迎的文章，包括文章，工具，视频等。在本篇中，肯定会有一些引起你注意的东西:-)

    1. ES2021：特性速递 — 早在今年 6 月，Ecma 大会就批准了 ES2021。 有关 ECMAScript 2022 草案规范 (opens new window) 工作仍在继续。但是这篇介绍 ES2021 新功能的帖子是我们今年最受欢迎的链接，挺令人惊讶的。 Axel Rauschmayer 博士还有一篇 关于 ES2021 的热门帖子 (opens new window)。

    2. 比较 Svelte 和 React — 两个框架一起使用了很长一段时间后，作者根据真实示例确定了他喜欢哪个。我怀疑 2022 年 Svelte 将继续向前迈进一大步。

    3. 新一代构建工具对比 — 新一代构建工具对比，包括 Snowpack、esbuild、Vite 和 wmr。

    4. Rust 是 JavaScript 未来 (opens new window) — 只能说这是一个意见，但日益增长的数据是很明显的。 Rust（以及 Go）现在大量用于替换可能在 JS 之前编写的 JavaScript 工具生态系统的部分（例如 Rome，SWC，Dink）。

    5. 2021 年 Web Workers 的趋势 (opens new window) — 网络是单线程的，但两年前 Surma 告诉我们为什么 要使用 Web Workers (opens new window)，尽管有困难，也要将多线程成为您的解决方案之一。

    6. 重新思考 JavaScript 的三元运算符 — 三元运算符 ? "true" : "false"……许多开发人员对三元运算符“持一点怀疑态度”，James 说：它的简洁性是否值得单独使用？它是你可以“信任”的东西吗？ James 还研究了该语言的未来替代方案。

    简化 React 身份验证 (opens new window) — 在本文中，我们在 React 程序中布置了一种新的身份验证（加入控制器和 SSO）的方法。

    # 📅 JavaScript 的 2021: 每月记事

    一月： JS 的 2020 总结 (opens new window) 出来了。和 Snowpack 3.0 (opens new window) 一样，React 的创建者 离开了 Facebook (opens new window)。

    二月： npm 7.0 (opens new window) 发布，如 Vite 2.0 (opens new window)。

    三月： Node.js 团队开始思考 未来 10 年 (opens new window)。同时，jQuery 3.6.0 (opens new window) 发布。V8 加快了发布时间 (opens new window)，发布了 ES2021 候选规范 (opens new window)。 v8 9.0 (opens new window) 已发布。Deno 公司成立 (opens new window)。

    四月： Node.js 16 (opens new window) 发布，Node 中的 ES module 开始起飞 (opens new window)。

    五月： Rome 创造了一家公司，开始筹钱，招聘员工 (opens new window)。

    六月： UNDICI 4 (opens new window) 发布并作为 Node.js 更好的替代 HTTP 客户端。React 18 (opens new window) 揭晓。 ES2021 已完全批准。

    七月： 低代码环境 Node-RED 2.0 (opens new window) 发布。

    八月： Wikimedia Foundation 选择了 (opens new window) Vue.js 作为其前端的框架。 Vue 3.2 (opens new window) 发布。 Ruby 在 Rails 团队中被认为是 Ruby 框架中 处理 JavaScript 的新方法 (opens new window)。

    九月： Node.js 获得了一个名为 Corepack (opens new window) 的实验包管理器（SIC）。

    十月： Node 17 (opens new window) 发布。 React 的新文档 (opens new window) 网站已亮相。 jQuery Mobile 已被推翻 (opens new window)。Parcel 2 (opens new window) 发布。Remix 项目持资金 300 万美元 (opens new window)。Gatsby 4 (opens new window) 发布。

    十一月： React 18 (opens new window) 进入测试版。Angular 13 (opens new window) 发布。Remix 开源了 (opens new window)。

    十二月： Create React App 5.0 (opens new window) 发布.

    # 🛠 2021 年的顶级代码 & 工具

    zx：用于编写更好的 shell 脚本的工具 (opens new window) — 和 bash 不一样，zx 提供了各种各样的样式来和你打交道。

    100 多个算法和数据结构的 JS 演示 (opens new window) — 许多常见算法的示例（例如，比特操纵，帕斯卡尔三角形，汉明距离）和数据结构（例如，链接的列表，尝试，图形）等。

    HTM 3.1.0：使用标准标记模板的 JSX 替代方案 (opens new window) — 这个资源已经有好几年了，但仍然在更新。类似 JSX 样式语法，在普通的 JavaScript（使用标记模板）中，也支持像 REST Spread 和 Reference 组件这样的内容。

    Tagify：优雅的标签输入组件 (opens new window) — 最好的例子就在这里， GitHub  仓库 (opens new window)。

    Typehole: VS 代码工具自动生成类型和接口 (opens new window) — 一个可帮助您自动创建运行时值的初始静态类型的扩展。

    Cheetah Grid: Web 最快的开源数据表？ (opens new window) — 我们之前没有遇到特别大的问题。如果你想自己尝试，这里有一个表内有 100 万条记录的实时演示。新版本也刚刚发布。
