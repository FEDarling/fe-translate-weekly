    Using Web Streams in Node.js (opens new window) — Streams provide a handy abstraction for cleanly working with streaming data but Web Streams (a.k.a. WHATWG Streams (opens new window)) are a different, more broadly supported beast to classic Node.js streams. (opens new window) Dr. Axel starts from the basics but goes into a lot of depth here - a handy resource.

    Dr. Axel Rauschmayer

    Alternatives to Installing npm Packages Globally (opens new window) — A topic that often trips folks up, also covered by the venerable doctor.

    Dr. Axel Rauschmayer

    (opens new window)

    Integrate eSignatures in Your App in Less Than a Week (opens new window) — Enable document signing from anywhere with HelloSign's eSignature functionality. Test the API for free.

    HelloSign, a Dropbox Company sponsor

    Puppeteer 15.0: Control Headless Chrome from Node (opens new window) — Playwright (opens new window) has taken a lot of the browser remote control attention in recent years, but Puppeteer, which focuses on Chrome, is still a great option. It now uses Chromium 103 and supports Node 18.

    Google

    Microvium: A Tiny JavaScript Engine for Microcontrollers (opens new window) — How tiny? Try 8.5 kilobytes tiny and using just 34 bytes of RAM when idle. JavaScript has never been a natural fit for such constraints but projects like this (and Elk (opens new window) or low.js (opens new window)) are doing interesting things (here’s how Microvium works (opens new window)) and this one has particularly good language support. GitHub repo (opens new window).

    Michael Hunter
